# **App Name**: CleanMadurai.AI

## Core Features:

- Role-Based Access & Authentication: Secure user registration and login system supporting five distinct roles (Corporation Commissioner, Ward Admin, Zone Admin, Worker, Citizen) with role-based profile storage and redirection to appropriate dashboards.
- Role-Specific Summary Dashboards: Intuitive dashboards tailored to each user role, displaying essential overview metrics, tasks, and performance indicators relevant to their responsibilities.
- Dynamic Task Management System: System to automatically generate new tasks based on sensor alerts or citizen requests, and allow administrators (Ward Admin, Zone Admin) to efficiently assign these tasks to specific teams or individual workers.
- Worker Task Execution & Verification: Workers can view, accept, reject, and mark tasks as complete. Includes a QR code scanning feature for verifying proper waste disposal at designated points, and an automatic task forwarding logic if tasks are unaccepted or rejected.
- Citizen Engagement & Reporting Portal: A portal for citizens to submit public complaints with image proof (e.g., overflowing dustbins) or request private waste collection/cleaning services, directing these to the relevant Zone Admins.
- Real-time Sensor Integration: Integration with IoT sensors (dustbin level, water leakage, toilet stock) to receive real-time alerts, which automatically trigger the creation and assignment of maintenance or cleaning tasks.
- AI-Powered Performance Advisor Tool: Provides Corporation and Ward Admins with contextual insights, performance trend analysis, and suggested actionable advice to optimize resource allocation and improve waste management efficiency across wards and zones.

## Style Guidelines:

- Primary brand color: A clean, professional medium blue (#3373CC). This color signifies reliability, efficiency, and clarity, reflecting the app's commitment to effective urban management. Its saturation provides good visual weight on a light background.
- Background color: A very light, subtle shade of blue (#F0F2F6), chosen to be visibly related to the primary color but heavily desaturated to create a calm and spacious interface that highlights content and calls to action.
- Accent color: A vibrant, bright cyan (#47E2EF). This color, analogous to the primary but distinct in hue, saturation, and lightness, will be used sparingly for interactive elements, highlights, and status indicators, providing visual pop and emphasizing key user actions or urgent alerts.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, tech-inspired, and impactful aesthetic, suitable for dashboard titles and key information. Body text font: 'Inter' (sans-serif) for its high readability, objective feel, and excellent legibility across various screen sizes and text lengths in reports and detailed views.
- Utilize a consistent set of clean, minimalist line-art icons that clearly represent waste management, reporting, locations, analytics, and user roles, ensuring quick comprehension and a modern aesthetic.
- Employ a modular, card-based layout for dashboards to clearly segment information for different roles. A responsive grid system will ensure optimal display across devices, facilitating efficient data scanning and navigation through hierarchical structures (e.g., Ward > Zone > Worker).
- Implement subtle and purposeful animations for state changes (e.g., task status updates, loading data), dashboard transitions, and feedback on user interactions (e.g., button clicks, successful form submissions). Animations should be smooth and contribute to a polished user experience without being distracting.
